# RangerBlock Just-Chat

**One-Click P2P Blockchain Chat Client**

Version: 1.1.0 | November 2025

---

## Quick Install

```bash
curl -fsSL https://raw.githubusercontent.com/davidtkeane/rangerplex-ai/main/rangerblock/just-chat/just-chat.sh -o just-chat.sh && chmod +x just-chat.sh && ./just-chat.sh
```

## Web Chat (Mobile/Browser)

Access from any browser:
```
https://davidtkeane.github.io/rangerplex-ai/rangerblock/just-chat/web-chat.html
```

## Features

- **One-Click Install** - Single script installs everything
- **Interactive Menu** - User-friendly interface
- **Auto Node.js Install** - Installs Node.js if not present
- **AWS Relay** - 24/7 uptime cloud server
- **Web Client** - Mobile-friendly HTML chat
- **Visual Fireworks** - Beautiful terminal output

---

## Terminal Commands

| Command | Description |
|---------|-------------|
| `./just-chat.sh` | Show interactive menu |
| `./just-chat.sh -c` | Start chatting |
| `./just-chat.sh -t` | Test connection |
| `./just-chat.sh -s` | Show network status |
| `./just-chat.sh -u` | Update to latest |
| `./just-chat.sh -h` | Show help |
| `./just-chat.sh -i` | Run install |
| `./just-chat.sh -getoffmymachine` | Uninstall |

---

## Chat Commands

| Command | Description |
|---------|-------------|
| `/help` | Show all commands |
| `/who` | List online users |
| `/nick <name>` | Change nickname |
| `/msg <user> <text>` | Private message |
| `/quit` | Leave chat |
