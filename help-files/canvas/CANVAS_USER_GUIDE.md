## üíæ Saving Your Work

### Auto-Save (Automatic!)

**Your work is automatically saved**:
- Saves every 30 seconds
- Stored in your browser (localStorage)
- Loads automatically when you return
- No action needed!

**Storage info**:
- Typical canvas: ~500KB - 2MB
- Browser limit: ~5-10MB
- Warning appears if storage is full

---

### Save as PNG

**High-quality image export**:
1. Click the **üíæ PNG** button
2. Or press `Ctrl+S` (or `Cmd+S` on Mac)
3. File downloads automatically
4. Filename: `rangerplex-canvas-[timestamp].png`

**Best for**:
- Sharing on social media
- Inserting into documents
- Archiving your work
- Transparent backgrounds (if supported)

---

### Save as JPG

**Smaller file size**:
1. Click the **üíæ JPG** button
2. File downloads automatically
3. Filename: `rangerplex-canvas-[timestamp].jpg`
4. 90% quality (great balance!)

**Best for**:
- Email attachments (smaller size)
- Web uploads
- Quick sharing
- When file size matters

---

### Clear Canvas

**Start fresh**:
1. Click the **üóëÔ∏è Clear** button
2. Confirm the action (this cannot be undone!)
3. Canvas returns to blank state
4. Auto-save is cleared

**Warning**: This action is permanent! Save your work first if you want to keep it!

## üÜò Troubleshooting

### Canvas won't load

**Solutions**:
1. Refresh the page
2. Clear browser cache
3. Check if JavaScript is enabled
4. Try a different browser

---

### Auto-save not working

**Check**:
1. Browser storage not full (check console)
2. Private/Incognito mode (localStorage disabled)
3. Browser permissions

**Fix**:
- Clear old saved canvases
- Use regular browser window (not private)
- Check browser settings

---

### Drawing is laggy

**Solutions**:
1. Close other browser tabs
2. Reduce canvas size
3. Clear undo history (draw, then clear)
4. Use a faster device if possible

---

### Touch not working

**Check**:
1. Device supports touch
2. Browser supports touch events
3. Canvas is in focus (click it first)

**Fix**:
- Update browser to latest version
- Try different browser
- Restart device
